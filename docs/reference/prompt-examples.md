# Prompt examples

The repository supplies the stack, style and workflow. Prompts can stay focused on product intent.

```text
增加收藏市场：未登录时保存在本地，登录后与服务端同步。首页和详情页都能收藏。
```

```text
增加我的持仓页面，支持按进行中、已结算筛选，并显示总投入、当前价值和盈亏。
```

```text
增加市场评论区。先做只读列表和分页，暂不允许发布，重点评审内容安全和刷量风险。
```

```text
把市场详情的趋势图增加 1D、1W、1M 时间范围，保持现有视觉语言。
```

Agent should create a task folder, run role reviews, update documentation and execute quality gates without the prompt repeating those requirements.
