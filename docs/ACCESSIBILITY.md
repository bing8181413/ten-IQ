# Accessibility

Target WCAG 2.2 AA.

- Use semantic headings, landmarks, links and buttons.
- Every input has a programmatic label.
- Focus is visible and follows visual order.
- Normal text meets 4.5:1 contrast; large text meets 3:1.
- Do not communicate market direction by color alone.
- Controls remain usable at 200% zoom and on narrow screens.
- Respect reduced motion.
- Run Storybook a11y and Playwright axe checks for UI changes.
